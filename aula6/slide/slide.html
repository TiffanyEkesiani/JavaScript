Aula 6: Organizando o Código com Funções

Criando blocos de código reutilizáveis e inteligentes.

O que é uma Função?

Uma função é um bloco de código nomeado, projetado para realizar uma tarefa específica. É como criar uma nova ferramenta ou um novo comando na linguagem.

Definir a Função (Criar a ferramenta): 
function saudacao() {
    console.log("Olá, mundo!");
}
Chamar a Função (Usar a ferramenta): 
saudacao(); // Saída: "Olá, mundo!"

Dando "Ingredientes" às Funções: Parâmetros

Parâmetros são variáveis que uma função recebe para poder trabalhar. Eles tornam as funções flexíveis e poderosas.

const nome = "Visitante";
function saudacaoPersonalizada( nome ) {
    console.log(`Olá, ${nome}!`);
}

saudacaoPersonalizada(nome);   // Saída: Olá, Visitante!
saudacaoPersonalizada("Ana");   // Saída: Olá, Ana!
saudacaoPersonalizada("Carlos"); // Saída: Olá, Carlos!

Recebendo Resultados: return

A palavra-chave return finaliza a execução da função e especifica um valor a ser "devolvido" para quem a chamou.

function somar(a, b) {
    return a + b; // Devolve o resultado da soma
}

const resultado = somar(5, 3); // resultado agora vale 8
console.log(resultado);
Uma função sem `return` devolve `undefined` por padrão.

Onde as Variáveis Vivem: Escopo

Variáveis declaradas com let e const dentro de uma função (entre { }) só existem dentro daquela função. Isso é chamado de escopo de bloco.

function calcular() {
    const numeroInterno = 100; // Só existe aqui dentro
    console.log(numeroInterno);
}

calcular(); // Saída: 100
console.log(numeroInterno); // ERRO: numeroInterno is not defined
